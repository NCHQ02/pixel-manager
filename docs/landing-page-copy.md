# Landing Page Copy

## Hero

OmniSignal Pixel Tracker

Audit your marketing pixels before the ad budget starts burning.

Inspect Meta, TikTok, GA4, Google Ads, Floodlight, and GTM events in real time
from a local Chrome extension.

Primary CTA: Install Chrome Extension

Secondary CTA: View Privacy Policy

## Use Cases

Audit Pixel Events

See which events fire, when they fire, which pixel ID receives them, and what
payload was sent.

Detect Duplicate Firing

Spot suspicious repeated events and separate diagnostics from meaningful user
actions.

Inspect Raw Payloads

Review event parameters, consent signals, purchase values, currencies, and
advanced matching fields without leaving the browser.

Export QA Reports

Generate an HTML audit report with detected platforms, expected-vs-actual
events, issues, and raw payload appendices for dev or client handoff.

## Trust Copy

Data stays local by default. OmniSignal has no backend service in v1 and redacts
likely plaintext sensitive values before local storage.

## Demo Storyboard

1. Open a checkout thank-you page.
2. Click the OmniSignal extension icon.
3. Trigger a test purchase or reload the page.
4. Show a Purchase event with missing `value` or `currency`.
5. Open Issues and export the HTML report.
